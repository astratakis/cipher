HOW TO RUN:

-> sudo make
This will create 2 executable files: [validate_api.exe, demo_program.exe].

-> ./validate_api.exe
This will run some tests for the implementations of the algorithms. The tests are predefined and there are NO random numbers involed.

-> ./demo_program.exe
This will run the demo program.
